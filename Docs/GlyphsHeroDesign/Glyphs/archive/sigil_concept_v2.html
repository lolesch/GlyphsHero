Spatial Zone Map — How Layers Stack Without Competing
Weapon — the center. The root sigil, always present.
Activator / Reactor — top-left corner of the rune. Small, distinct. Mutually exclusive, so they share one zone.
Amplifier — branches radiating outward from the weapon core. Each amplifier adds a branch arm.
Payload — small symbol sitting at the tip of an amplifier arm, or at the outer edge if unanchored.
Condition — a filter glyph overlaid on whichever zone it gates. Visually suggests "stuff passes through here."
Converter — no new shape. Changes the color of what it converts — weapon glyph color shifts if damage type changes; an amplifier arm shifts color if it's being converted.
